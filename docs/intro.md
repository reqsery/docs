# Welcome to Revro Documentation

This is the central documentation hub for all projects built and maintained by [reqsery](https://github.com/reqsery). Whether you're a Roblox developer, a Discord server owner, or a Minecraft modder, you'll find everything you need here.

---

## Projects

### Revro — AI Platform for Roblox & Discord

[Revro](https://revro.dev) is an AI-powered creation platform that lets you generate Roblox scripts, design UI elements, and configure Discord servers — all from your browser or directly inside Roblox Studio.

**Key features:**
- Generate production-ready Lua scripts in seconds
- Create complex UI layouts with AI
- Design and deploy Discord server setups automatically
- Generate game assets and images (Pro/Studio)
- Direct Roblox Studio plugin integration

➡️ [Get started with Revro](/revro/getting-started)

---

### MC Mod Porter — Minecraft Mod Migration Tool

[MC Mod Porter](https://github.com/reqsery/mc-mod-porter) is an open-source CLI tool that automatically ports Minecraft Java Edition mods across versions — from 1.16 all the way to the new 26.x calendar versioning format.

**Key features:**
- Automated API change detection and patching
- Supports both Fabric and NeoForge mod loaders
- Dry-run mode to preview changes before applying
- AI-assisted workflow support (Claude Code, Cursor, Windsurf)
- Comprehensive knowledge base for every version hop

➡️ [Get started with MC Mod Porter](/mc-mod-porter/installation)
