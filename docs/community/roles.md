# Server Roles

Here's a breakdown of all roles in the [Revro Discord server](https://discord.gg/vV2USr9phF) and what each one means.

---

## Staff Roles

### Owner
The server owner. Has full control over all server settings, channels, roles, and bots. Currently held by the project lead (@reqsery).

### Admin
Admins assist the owner with server management, can add/remove bots, and handle escalated issues and bans. Appointed by the Owner.

### Moderator
Moderators enforce server rules, manage tickets, and handle day-to-day moderation. They can kick/ban members and manage messages.

### Trial Mod
New moderators in training. Trial Mods can respond to tickets and help members but have limited moderation powers. They are evaluated before being promoted to Moderator.

### Helper
Community members who regularly help others with Revro, MC Mod Porter, or Discord setup questions. Helpers are trusted community members, not official staff.

---

## Community Roles

### Beta Tester
Members who have access to new features before they're publicly released. Beta Testers are selected periodically based on activity and contributions.

**How to get it:** Be active in the community and express interest to a Moderator or Admin. Invitations are sent when new beta testing rounds open.

### Member
The default role assigned to everyone when they join the server. Members have access to all standard channels.

---

## Role Hierarchy

```
Owner
└── Admin
    └── Moderator
        └── Trial Mod
            └── Helper
                └── Beta Tester
                    └── Member
```

---

## Can I Get a Role?

| Role | How to Get It |
|------|--------------|
| Owner | Project owner — not available |
| Admin | Appointed by Owner — not available |
| Moderator | Invited by Admin — not available on request |
| Trial Mod | Invited by Admin — not available on request |
| Helper | Invited based on community activity |
| Beta Tester | Invited when beta rounds open |
| Member | Automatic on join |

If you're interested in becoming a Helper or Beta Tester, the best thing to do is be active, helpful, and constructive in the server. Staff will notice.
