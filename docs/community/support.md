# Getting Support

There are several ways to get help depending on the type of issue you have.

---

## Support Channels

### Discord Server (Fastest)

The [Revro Discord server](https://discord.gg/vV2USr9phF) is the fastest way to get help. The community and staff are active daily.

**How to use it:**
1. Join the server at [discord.gg/vV2USr9phF](https://discord.gg/vV2USr9phF)
2. Open a ticket in the **#support** channel for private help
3. Or ask in the relevant public channel for quick questions

The [FAQ Bot](/discord-bots/faq-bot) monitors tickets and may respond automatically within seconds.

---

### Email Support

For account, billing, and sensitive issues:

| Email | Use For |
|-------|---------|
| [support@revro.dev](mailto:support@revro.dev) | Account issues, billing, credits, refunds |
| [reqsery.dev@gmail.com](mailto:reqsery.dev@gmail.com) | Business inquiries, partnerships |

Response time: typically within 24–48 hours on business days.

---

### GitHub Issues

For **bugs** and **feature requests** on open-source projects:

| Project | Link |
|---------|------|
| MC Mod Porter | [github.com/reqsery/mc-mod-porter/issues](https://github.com/reqsery/mc-mod-porter/issues) |

When opening an issue, include:
- A clear title describing the problem
- Steps to reproduce
- Expected vs. actual behavior
- Your environment (OS, Java version, Minecraft version)

## What Kind of Issue Do You Have?

| Issue Type | Best Channel |
|-----------|-------------|
| Revro credits / billing | [support@revro.dev](mailto:support@revro.dev) |
| Can't log in / lost API key | [support@revro.dev](mailto:support@revro.dev) |
| AI generation errors | Discord ticket or support email |
| Roblox plugin not working | Discord ticket |
| MC Mod Porter bug | GitHub Issues |
| MC Mod Porter question | Discord server |
| Feature request | Discord server or GitHub Issues |
| General question | Discord server |
| Partnership / business | [reqsery.dev@gmail.com](mailto:reqsery.dev@gmail.com) |
